# Luta Feminina 2D (Kotlin + Jetpack Compose)

Projeto Android de jogo 2D de luta em Compose com:

- 5 personagens femininas brasileiras (18 a 20 anos), estilos e poderes diferentes.
- Seleção de roupa com opções **Biquíni** e **Lingerie**.
- 3 cenários: **Cidade**, **Praia** e **Casa**.
- Sistema de combate com soco, chute e poder especial.
- Barra de energia especial que carrega ao **segurar o botão** de carga.

## Como abrir no Android Studio

1. Abra a pasta do projeto no Android Studio.
2. Aguarde sincronização do Gradle.
3. Rode no emulador/dispositivo Android.

> Se o Gradle Wrapper não estiver disponível no ambiente local, gere com `gradle wrapper` usando Java 17+.
